# WebTech Practice

Welcome to my Web Technology Practice Repository. This repository contains all the programs, practicals, assignments, and exercises completed during my Web Technology coursework. It serves as a record of my learning journey and helps me track my progress as I develop my web development skills.

## About This Repository

This repository is organized according to the daily topics taught in class. Each folder represents a specific day of learning and contains the programs, examples, and exercises covered during that session.

The purpose of this repository is to:

* Maintain a structured collection of classwork and practical exercises.
* Track learning progress throughout the course.
* Practice version control using Git and GitHub.
* Build a strong foundation in web development technologies.
* Create a reference resource for future revision and projects.
